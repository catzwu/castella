Create a site for karaoke song queueing and voting. 

Core functionality
* Create a room with a join code to share. No login required, just a username
* Once created, room persists for a day at the same login code, no matter how many people are in it. 
* Search up songs to add to the queue
* Upvote and downvote songs. Only one upvote/downvote per song per person. Votes can be changed or retracted
* Display who added a song.
* Ability to edit (reorder, delete) the queue by anyone.
* Display an already played section (just for display)

Additional functionality
* Recommend songs based on what is in the queue already, users can add from recommendations
* Automatically search up lyric videos on Youtube or Spotify and play them when next in queue. Have a display mode
* Have a song "waiting room" sorted by upvotes. Songs are added into the waiting room. Songs with enough upvotes get automatically added to the queue.
* Settings section to control e.g. how many upvotes needed to get automatically added to the queue

Style
* Modern, minimalist.
